elasticsearch - Search for web developers

  • Java, based on Apache Lucene
  • Scales out
  • Every interaction can be done via HTTP & JSON
    • Creating/updating/deleting documents into an index
    • Creating/updating/deleting indexes
    • Search queries
  • References: Mozilla, StumbleUpon, Sony, Infochimps, Assistly

Data representation in JSON

{
	"id": "1",
	"name" : "MacBook Air",
	"price": 1099,
	"descr" : "Some lengthy never-read description", 
	"attributes" : {
		"color" : "silver",
		"display" : 13.3,
		"ram" : 4
	}
}

Installation needs two minutes

  • Download
    https://github.com/downloads/elasticsearch/elasticsearch/elasticsearch-0.18.2.zip
  • Unzip
    unzip elasticsearch-0.18.2.zip
  • Run
    elasticsearch/bin/elasticsearch
  • Check by connecting with your browser to http://localhost:9000

Index your product

curl -X PUT localhost:9200/products/product/1 -d '{
	"id": "1",
	"name" : "MacBook Air",
	"price": 1099,
	"descr" : "Some lengthy never-read description", 
	"attributes" : {
		"color" : "silver",
		"display" : 13.3,
		"ram" : 4
	}
}'

JSON as query language - Normal query

http://localhost:9200/products/product/_search 
  • { "query" : { "term" : { "name": "MacBook Air" }}}
  • { "query" : { "prefix" : { "name": "Mac" }}}
  • Documents can also be read via their id: http://localhost:9200/products/product/1
Queries can be issued via curl or via HttpRequester

JSON as query language - Range query

Useful for price, date and string ranges

{ "query" : { "range" : { 
	"price" : { "from" : 1000, "to": 2000 }
} } }

JSON as query language - Pagination & Sorting

  • Pagination: 
    { from: 0, size:10, 
  "query" : { "term" : { "name": "MacBook Air" }}}
  • Sorting: 
    { "sort" : { "name" :  { "order": "asc" } },
      "query" : { "term" : { "name": "MacBook Air" }}}

Far more features!

  • Faceted queries, sorting, highlighting, filters
  • Mapping/Analyzers
  • Geolocation support
  • Boosting: Applying different weights to documents or fields
  • Custom scripting (this is where your own integration beats out all others)
  • Rivers: Push/pull data from arbitrary sources
